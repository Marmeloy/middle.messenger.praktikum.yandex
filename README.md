## Инструменты
- TypeScript
- Node Express
- <b>Шаблонизатор:</b> Pug
- <b>Сборщик:</b> Parcel
- <b>Code Style:</b> ESLint & StyleLint

## Команды

Развёртывание проекта:
```
npm i
```

Сборка:
```
npm run build
```

Запуск (dev):
```
npm run dev
```
Проект запустится по URL:
```
http://localhost:3000
```
